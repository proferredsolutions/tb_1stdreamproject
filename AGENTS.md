# Agent Instructions for 1stdreamproject

Welcome to the 1stdreamproject. This project aims to build a notification system as outlined in the initial configuration.

## Project Overview
- **Name:** 1stdreamproject
- **Core Functionality:** Resource monitoring and notification system.
- **Primary Language:** Python

## Coding Standards
- Follow PEP 8 guidelines for Python code.
- Use type hints for all function signatures.
- Write docstrings for all modules, classes, and functions using the Google style.
- Ensure all new features are accompanied by unit tests.

## Development Workflow
- **Dependencies:** Manage dependencies in `requirements.txt`.
- **Testing:** Use `pytest` for running tests.
- **Configuration:** Use `config.yaml` for application settings.

## Instructions for Agents
- Always check `config.yaml` for environment-specific settings.
- Before submitting, ensure all tests pass and `AGENTS.md` is updated if any new conventions are established.
- Maintain a clean and modular codebase in the `src/` directory.
